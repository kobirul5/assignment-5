Important FAQ

A.What is the Document Object Model (DOM)?

The Document Object Model (MOM) is a programming interface that respresents the structurre of an HTML Documnet as a tree of object.

B. How do you select an element from the DOM?
  • getElementById("id")
  • getElementsByClass("class")
  • getElementsByTagName("tag")
  • querySelector("selector")
  • querySelectorAll("selector")
C. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent gelement, allaowing it to handle events from child elements as they bubble up

D. How do you manipulate an element's attributes and styles using the DOM?
  • Element.SetAttribute('attr', 'value')
  • Element.getAttribute('attr')
